Instrucciones para rellenar el formulario

El Nombre solo puede contener letras(puede haber mas de uno)

El Apellido tiene las mismas instrucciones que el nombre

El email no puede contener caracters extraños(exceptuando la @ y el .)

La contraseña debe contener como mínimo 8 caracteres, entre ellos una mayúscula una minúscula, un número y un caracter especial

Confirmar Contraseña debe ser igual a la contraseña añadida anteriormente

El telefono debe tener 9 caracteres, y no puede haber espacios entre ellos

el DNI debe tener 8 numeros y una letra, sin espacios entre ellos

Razon social hace referencia al nombre de la empresa, este se valida igual que el nombre

presiona ESC para salir
